The Basic Language of WEB: HTML

Posted by Aizaz Naqvi on Friday, July 14th 2023

All the modern web applications and mobile applications are built using three fundamental technologies: HTML, CSS, and javascript. these are the languages of the web.

In this Post, Lets focus on HTML. We will learn what is HTML all about, and why you too should learn it.

What is HTML?

HTML stand for HyperText Markup Language. Its's a markup language that web developers use to structure and describe the content of a webpage (not a programming language).

HTML consist of element that describe different types of content: Paragraph, links, heading, images, video, etc. Web browsers understand HTML and render HTML code as websites.

in HTML each element made up of three parts

  1. Opening tag
  2. The closing tag
  3. The Actual Element

Why Should you learn HTML?

There are countless reasons for learning the fundamental language of the web. Here are 5 of them.

  • To be able to use fundamental Web dev language
  • To hand craft beautiful website instead of relaying on tools like wordpress or wix
  • To build web applications
  • To impress friends
  • To have fun
